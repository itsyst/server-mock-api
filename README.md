# Server Mock API

A Fake REST API using json-server.

> This could help you shape your frontend application without having to rely on any backend services.

## Install

```
  npm install
```

## Generate data

```
  npm run seed
```

Available endpoints would be:

- GET `/products`
- POST `/products`
- GET `/products/:id`
- PUT `/products/:id`
- DELETE `/products/:id`


## Run Server

```
  npm run server
```
